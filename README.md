# Personal Nutrition Recommendation System

A Python-based nutrition recommendation system that provides personalized dietary suggestions.

## Prerequisites

- **Python 3.9 or higher**
- pip (Python package manager)
- Virtual environment support

Verify Python installation:

```bash
python --version
```

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/GENERAL-PRIME/personal-nutri-recommend-sys.git
cd personal-nutri-recommend-sys
```

### 2. Create a Virtual Environment

**Windows:**

```bash
python -m venv venv
```

**Linux / macOS:**

```bash
python3 -m venv venv
```

### 3. Activate Virtual Environment

| Operating System | Shell          | Command                     |
| ---------------- | -------------- | --------------------------- |
| Windows          | Command Prompt | `venv\Scripts\activate`     |
| Windows          | PowerShell     | `venv\Scripts\Activate.ps1` |
| Linux / macOS    | Bash / Zsh     | `source venv/bin/activate`  |

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

## Usage

### Run the Program

**Windows:**

```bash
python main.py
```

**Linux / macOS:**

```bash
python3 main.py
```

### Deactivate Virtual Environment

When finished, deactivate the virtual environment:

```bash
deactivate
```
